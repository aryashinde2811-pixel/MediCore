# 🏥 MediCore

A smart healthcare UI prototype designed to visualize and simulate AI-based resource allocation in hospitals.

---

## 🚀 Problem Statement

Hospitals often struggle with:

* Overcrowding
* Poor resource allocation (beds, ICU, doctors)
* Delays in emergency response

These inefficiencies can impact patient outcomes and hospital performance.

---

## 💡 Solution

**MediCore** provides an interactive interface that demonstrates how AI can:

* Prioritize patients based on severity
* Allocate hospital resources efficiently
* Improve emergency handling workflows

> ⚠️ Note: This is currently a **frontend prototype** demonstrating the concept and UI/UX of the system.

---

## 🔥 Core Features (MVP)

* 🧠 **Patient Prioritization Simulation**
  UI-based logic to categorize patients by urgency

* 🏥 **Resource Allocation Interface**
  Visual representation of beds, ICU, and doctor availability

* 🚑 **Emergency Case Highlighting**
  Critical patients are emphasized for faster response

* 📊 **Dashboard View**
  Displays system overview and resource distribution
